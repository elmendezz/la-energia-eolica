Nuestro Proyecto

Hemos diseñado y construido un modelo funcional de un generador eólico (Digital). Utilizando un motor de corriente continua, aspas diseñadas para capturar el viento y un circuito simple, este "abanico" es capaz de generar suficiente energía para encender una luz LED, sirviendo como nuestro logo y la prueba de concepto de nuestro trabajo.

¿Cómo Funciona?

¿Cómo Funciona?

1

Captura de Viento

El viento incide sobre las aspas, diseñadas con una forma aerodinámica para maximizar la fuerza de rotación.

2

Generación

Las aspas están conectadas a un pequeño motor que, al girar, funciona como un generador, convirtiendo la energía mecánica en eléctrica.

3

Iluminación

La corriente eléctrica generada viaja a través de un cable hasta una bombilla LED, que se ilumina demostrando la conversión de energía.

Materiales Utilizados

Para construir nuestro generador, utilizamos componentes simples y fáciles de encontrar, demostrando que la ciencia puede ser accesible para todos.

Motor DC

El corazón del proyecto. Funciona a la inversa para actuar como generador.

Diodo LED

Nuestro indicador visual, que se ilumina con la energía generada.

Cables y Conectores

Para crear el circuito y transportar la electricidad desde el motor hasta el LED.

Ver Impacto

Impacto y Sostenibilidad

Nuestro proyecto no solo demuestra la viabilidad de la energía eólica a pequeña escala, sino que también subraya la importancia crítica de las fuentes de energía renovable para un futuro sostenible. Cada pequeña iniciativa que promueve la energía limpia contribuye a reducir nuestra huella de carbono y a combatir el cambio climático.

Al entender cómo funciona un generador eólico, podemos apreciar mejor el potencial de la naturaleza para proveer energía sin agotar recursos ni contaminar el medio ambiente. Es un paso hacia la conciencia ecológica y la innovación.

Ver Conclusión

Conclusión

Con este proyecto, hemos visto cómo el viento puede convertirse en electricidad para encender una luz. Es una forma sencilla de entender la energía eólica y cómo podemos usar recursos naturales para cuidar nuestro planeta.

Esperamos que este proyecto te inspire a explorar más sobre las energías renovables y cómo podemos construir un futuro más sostenible. ¡El viento nos muestra el camino!
